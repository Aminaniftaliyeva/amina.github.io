Project Overview

This personal website presents me as a creator and highlights my journey with projects. This is my third project, developed in Cursor, and it represents a significant milestone in my learning journey. Through this website, I showcase my work, share my accomplishments, and demonstrate my growth as a developer and creator.

Projects

This website features my three main projects, each representing a different stage of my learning journey:

  • How Does Hardware Works? - My first project exploring logic gates and hardware fundamentals through hands-on breadboard experiments.
  • Hour of Code - My second project where I collaborated with Team 43 to teach AI literacy to 6th grade students at Lachin School No.45 in Bilajari.
  • Personal Website - My third project, which you are currently viewing. This website itself is a demonstration of my ability to create, manage, and present my work professionally.

Development Process

This website was created using Cursor, an AI-powered code editor that helped me build and refine the project. The development process involved several key steps:

Using Cursor

I developed this entire website in Cursor, leveraging its AI capabilities to help with code generation, debugging, and optimization. Cursor allowed me to work more efficiently while learning best practices in web development.

Connecting to GitHub

I connected this project to GitHub to manage version control and showcase my work. Setting up the repository, initializing Git, and learning to use version control was an important part of this project.

Uploading Projects

I carefully uploaded all my projects to GitHub, organizing them properly and ensuring each project was well-documented. This process taught me the importance of proper project organization and version control.

Writing Prompts Independently

One of the most valuable aspects of this project was learning to write clear and effective prompts. I wrote all prompts by myself, carefully crafting instructions that would help me achieve the desired results. This skill has proven essential in working with AI tools and communicating my ideas effectively.

Learning to Write Prompts

Through this project, I learned how to write prompts clearly and effectively. I discovered that good prompts require:

  • Clear and specific instructions
  • Well-structured requirements
  • Context about what I want to achieve
  • Attention to detail in describing desired outcomes

This skill has not only helped me in this project but will be valuable in all my future work with AI-assisted development tools.

This website was built using HTML, CSS, and JavaScript to create a modern, interactive web experience. The project was developed in Cursor and is hosted on GitHub, demonstrating my ability to work with modern development tools and version control systems.

Conclusion

By creating this personal website, I proved to myself that I can independently manage projects, connect them to GitHub, and write clear prompts. I learned how to present my work professionally and gained confidence in my ability to grow as a creator. This project represents not just a website, but a journey of learning, self-discovery, and professional development.
